Tested on my personal M2 Macbook Air only, it uses ADB for input but the screenshots are still native so you need the game visible. it uses an apple script to properly resize/relocate the bluestacks air window, I'm not sure if this is reproducible on another machine I haven't tested.

I play this screensharing from my main Mac, I think the resolution of the laptop becomes 1080p if that helps.

Open the OCR region adjuster if fine adjustments are needed for the matching templates.

Execute mode will actually carry out the turn, if you leave it on Check Only it will do the analysis and spit out what it plans to do on the bottom of the planned actions text box. You can then copy paste that into some AI assistant to help modify the bot logic. Continue button to proceed and execute that turn only.

It's really hacky i'm sorry, just needed something to bang out runs on MANT. it's really dumb and slow. I think we're at least 3-5 months behind so there are probably lots of things that were fixed in the mean time.

Run the usual command to start the python file, then hit start bot on the operator console. the rest should be self explanatory.
